HOW IT WORKS.


This method uses the Sia and Solana Blockchains to create a highly censorship resistant website.


"I'm a user who has a web site that is accessible from siasky.net"


I have my TLD pointing to a TXT record with the Sia hash directly on the handshake blockchain.


I want my site to be even more resilient so I will deploy the site using Media Network.


In this example I can use the URL of an image that I want to always be able to call into a web page. https://vcdn.hns.siasky.net


But, I don't want to host my own sky portal or rely on portals to always be alive and active 24/7.


To solve this problem I cached the above URL to the Media Network decentralized CDN powered by Solana Blockchain.


Once the transaction finalized it spit out this handshake hostname: https://3jgkxxtz.dcdn

Frequently Asked Questions

Got a question? We've got answers. If you have some other questions, contact us by clicking this link to hnschat/

How long will my .dCDN link be enabled?

Blockchain technology is mostly experimental. In essence, your links should live on forever as long as the Solana blockchain exists. We create fresh wallets on a random basis for the URL/IP deployments, and never save the keys or seeds. This allows the wallet to fall into the deep dark crypto depths never to be found or accessed again. Each time we cache a URL/IP onto the decentralized CDN Network we pay a small amount of Solana in order for the content to be cached and to create the .dCDN hostname. Once it's deployed, it's deployed for good.

Why use this middle-man service? Can I use Media.Network directly?

This service is offered as a convenience for the Web3 community. You can use Media.Network directly without the need to pay 10 HNS. With us, you pay HNS to help put more funds back into developing the handshake community. Plus, if you did want to do it yourself, you will need to own 100 Media Tokens and pay Solana Gas Fees in order to submit a URL/IP to be cached on the Blockchain. Not to mention the exchange and withdrawal fees that you will also likely have to pay as well. If you use the Triton Private API you can deploy URLs/IPs using Solana. The other upside to using this service is that you never have to actually go through the process or take the action of creating a Solana wallet or submitting the transaction.

Do you sell or share my data with any other parties?

You can choose to publicly display your .dCDN link on our website, otherwise it is kept private. We do not share or sell information with anyone.

Is https:// supported on .dCDN URLs?

Yes, you can use https via DANE.

Can I add a custom port to access my URL?

Definitely! Yes you can. Please be advised this feature is experimental.

Can I request refund?

Their are no refunds once we deploy your URL/IP to the blockchain.
